# Parkinson’s Disease Detection using Multimodal Neuroimaging Data (CIS 5200 Fall 2025, Group 44)

This repo implements two single-modality baselines and two multimodal deep learning models for Parkinson’s disease (PD) detection using paired DaTSCAN and T2‐weighted sMRI from PPMI.

Note: PPMI data is not included in this repo. Please refer to the [PPMI website](https://www.ppmi-info.org/access-data-specimens/download-data) for data access.
